\subsection{Detailed Description}
{\bf Obit\-DCon\-Clean\-Bm\-Hist}{\rm (p.\,\pageref{structObitDConCleanBmHist})} class function definitions. 

This class determines the maximum beam sidelobe exterior to a given x or y distance. This class is derived from the {\bf Obit}{\rm (p.\,\pageref{structObit})} base class.
