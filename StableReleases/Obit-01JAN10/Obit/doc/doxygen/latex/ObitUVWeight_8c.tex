Default = 0.4. \item \char`\"{}Max\-Baseline\char`\"{} OBIT\_\-float scalar = maximum baseline length in wavelengths. Default = 1.0e15. \item \char`\"{}start\-Chann\char`\"{} OBIT\_\-long scalar = first channel (1-rel) in uv data to grid.
